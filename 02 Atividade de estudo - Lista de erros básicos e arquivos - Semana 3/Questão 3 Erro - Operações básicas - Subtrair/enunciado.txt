Escreva um programa para calcular quantas vezes é possível subtrair 3 de N. O usuário deve entrar com o N inteiro positivo.

Para isso, utilize o código fornecido. Baixe-o, faça as correções necessárias e submeta para avaliação.

Entradas:

Número inteiro positivo.
Saídas:

Número de vezes que é possível subtrair 3 do número.
Exemplo de Entrada:

15
Exemplo de Saída:

5
Exemplo de Entrada:

22
Exemplo de Saída:

7
